# Agora

Custom plugin marketplace for Claude Code.

Plugin authors maintain their own repos with semver releases. Agora holds a curated index (`plugins.json`) and an `agora` skill that translates it into Claude Code's `marketplace.json`, plus tooling for setup, version bumps, and update notifications.

Status: **design phase**. Implementation pending. See the design spec in the atelier repo: `docs/agora-design.md`.

## Naming

Greek `agora` = public marketplace and gathering place. Matches the convention of related sibling projects (memex, atelier) — name a place, not a function.
